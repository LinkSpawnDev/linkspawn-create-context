# linkspawn-create-context — agent entry point

Read `.context/AGENTS.md` first. It is the operating contract for all AI agents in this repository — guardrails, definition of done, and interaction protocols.

Then follow the context cascade in `.context/README.md`:
`AGENTS.md` → `PROJECT_STATE.md` → `ARCHITECTURE.md` → `DECISIONS.md` → `LESSONS.md`

Update `.context/PROJECT_STATE.md` before ending every session.
